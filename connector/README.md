# אוצר · מחבר בנק מקומי (Local Bank Connector)

חיבור **אמיתי ואוטומטי** לחשבון הבנק וכרטיסי האשראי שלך — לצורך דמו על נתונים אמיתיים.
הכל רץ **על המחשב שלך בלבד**. הסיסמאות שלך לא יוצאות מהמחשב ולא נשמרות בשום מקום.

מבוסס על הספרייה הפתוחה [`israeli-bank-scrapers`](https://github.com/eshaham/israeli-bank-scrapers) — אותה טכנולוגיה שמפעילות אפליקציות פיננסיות ישראליות (Caspion, Moneyman) לחשבון האישי.

## מה נתמך
- **בנקים:** הפועלים, לאומי, מזרחי טפחות, דיסקונט, מרכנתיל, אוצר החייל, יהב, וואן זירו ועוד
- **כרטיסי אשראי:** ויזה כאל, מקס (לאומי קארד), ישראכרט, אמריקן אקספרס

## איך מריצים (פעם אחת, ~3 דקות)

צריך **Node.js 18+** מותקן ([הורדה](https://nodejs.org)).

```bash
cd connector
npm install      # מוריד את המנוע + דפדפן פנימי (פעם אחת, ~1–2 דק')
npm start        # מפעיל את השרת המקומי
```

עכשיו פתח בדפדפן:

```
http://localhost:8788
```

זו אפליקציית אוצר המלאה. לחץ **תנועות → "חבר את הבנק שלך" → "חיבור חי אוטומטי"**,
בחר את הבנק/כרטיס, הזן את הפרטים שלך — והתנועות האמיתיות נמשכות ומסתווגות אוטומטית.

## אבטחה
- 🔒 **הכל מקומי.** השרת רץ על `localhost` שלך. אין שרת ענן, אין צד שלישי.
- 🔒 **הסיסמאות לא נשמרות.** הן משמשות רגע אחד להתחברות ומיד יוצאות מהזיכרון. אין קובץ, אין לוג.
- 🔒 **קוד פתוח.** אפשר לקרוא כל שורה ב-`server.js` וב-`israeli-bank-scrapers`.
- ⚠️ אל תריץ את זה על שרת ציבורי בלי אבטחה. זה נועד לרוץ על המחשב שלך.

## למשקיעים — ההקשר
זהו **דמו על נתונים אמיתיים** בגישת screen-scraping מקומית (הדרך שבה סטארטאפים פיננסיים ישראליים מדגימים לפני קבלת רישיון). **הגרסה לייצור** תתחבר דרך **Open Banking בתקן בנק ישראל** עם רישיון נותן שירות מידע פיננסי — כמפורט בסעיף 7 ב-PRD. הדמו הזה מוכיח שהמוצר עובד; הרישיון הופך אותו למוצר המוני.

## בעיות נפוצות
- **"ההתחברות נכשלה"** — ודא שם משתמש/סיסמה; חלק מהבנקים חוסמים אחרי כמה ניסיונות.
- **התקנה נתקעת** — ה-`npm install` מוריד דפדפן פנימי (Chromium); צריך חיבור אינטרנט טוב פעם אחת.
- **וואן זירו / בנקים עם OTP** — דורשים קוד חד-פעמי; ראה תיעוד israeli-bank-scrapers.
