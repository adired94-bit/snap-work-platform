# 🚀 איך להפעיל את האפליקציה

יש שתי דרכים. **דרך א' מומלצת** — מקבלים קישור קבוע באינטרנט בלי להתקין כלום.

---

## דרך א' — פריסה לענן (קישור קבוע, ללא התקנות) ⭐

תקבל כתובת כמו `https://chart-analyzer.onrender.com` שתפתח מכל מכשיר.

### שלב 1 — הרשמה ל-Render
1. כנס ל-https://render.com
2. לחץ **Get Started** → **Sign in with GitHub** (התחבר עם אותו חשבון GitHub שלך)
3. אשר את ההרשאות

### שלב 2 — יצירת השירות
1. בלוח הבקרה לחץ **New +** → **Blueprint**
2. בחר את המאגר **snap-work-platform**
3. Render יזהה אוטומטית את הקובץ `render.yaml` — לחץ **Apply** / **Create**

### שלב 3 — הוספת מפתח ה-API
1. אחרי היצירה, היכנס לשירות → לשונית **Environment**
2. לחץ **Add Environment Variable**:
   - Key: `ANTHROPIC_API_KEY`
   - Value: המפתח שלך מ-https://console.anthropic.com/settings/keys
3. שמור — Render יבנה ויפרוס אוטומטית (1-2 דקות)

### שלב 4 — מוכן!
למעלה בעמוד תופיע הכתובת (למשל `https://chart-analyzer.onrender.com`).
פתח אותה — והאפליקציה עובדת מכל מקום. 🎉

> 💡 בתוכנית החינמית השירות "נרדם" אחרי חוסר פעילות; הכניסה הראשונה אחרי הפסקה תיקח ~30 שניות להתעורר. זה נורמלי.

## דרך ב' — הרצה במחשב Windows (מקומית)

אם אתה מעדיף להריץ במחשב שלך:

1. **התקן Python** (פעם אחת): https://www.python.org/downloads
   ⚠️ בהתקנה סמן ✅ **"Add python.exe to PATH"**
2. **הורד את הפרויקט**: בדף המאגר ב-GitHub לחץ על הכפתור הירוק **Code** → **Download ZIP**, וחלץ את הקובץ.
3. בתוך התיקייה שחולצה, **לחץ פעמיים על `start_windows.bat`**.
   - בפעם הראשונה הוא יתקין ספריות ויבקש את מפתח ה-API — הדבק אותו והקש Enter.
   - הדפדפן ייפתח אוטומטית בכתובת http://localhost:5000

זהו. בפעמים הבאות פשוט לחץ שוב על `start_windows.bat`.
